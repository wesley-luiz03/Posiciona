# Posiciona — Site Institucional

Site institucional desenvolvido para a **Posiciona**, uma empresa focada em posicionamento digital, estratégia e produção de conteúdo para empresas.

O objetivo do projeto é apresentar os serviços da empresa, seus diferenciais, portfólio de entregas e planos de atuação de forma clara, moderna e profissional.

---

## 🚀 Demonstração

🔗 Acesse o site online:  
https://posiciona.vercel.app/

---

## 🎯 Objetivo do Projeto

Criar um site institucional que:

- apresente os serviços da empresa
- fortaleça a percepção de autoridade da marca
- facilite o contato com potenciais clientes
- demonstre exemplos de entregas e resultados
- seja responsivo para todos os dispositivos

---

O projeto foi desenvolvido utilizando:

## ⚙ Funcionalidades

O site possui:

- menu de navegação responsivo
- animações suaves de interface
- botão flutuante de contato via WhatsApp
- seção de serviços estratégicos
- apresentação de portfólio visual
- prova social (depoimentos)
- planos de serviços
- redirecionamento inteligente para contato
- modal de confirmação de redirecionamento
